Bootstrap adalah framework CCS, HTML serta JavaScript yang memiliki fungsi untuk dapat mendesain sebuah web saja yang responsif sehingga saat diakses akan lebih mudah dan cepat. Framework ini diciptakan mulai tahun 2011 oleh Mark Otto dan juga Jacob Thornton dari Twitter. Itulah salah satu alasan Bootstrap ini dulunya dinamakan Twitter Blueprint. Subscribe ini merupakan framework gratis yang bersifat terbuka atau open source. Bootstrap adalah framework yang menawarkan berbagai kemudahan yang membuat Anda tidak perlu di komponen mulai dari nol lagi.
